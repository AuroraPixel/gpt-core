# Java Langchain使用（LangChain4j）
## LangChain4j介绍
LangChain4j 于 2023 年初开发，旨在填补 Java 生态系统中 LLM 库的空白，融合了 LangChain、Haystack、LlamaIndex 等的思想和创新元素。
## 核心功能特点:
统一API管理：支持10多个LLM提供商、向量数据的存储、简化服务开发
全面工具箱：提示词模版、多模态、AI调用工具、Agent代理、RAG搜索
